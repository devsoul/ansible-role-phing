Phing Ansible role
=========

If you are a PHP developer and you use Phing.

Requirements
------------
  - `php > 5.2` 

Role Variables
--------------
None

Dependencies
------------
None

Example Playbook
----------------
    - hosts: php
      roles:
        - devsoul.phing

Check if it's installed:
------
    phing --version
